# Bus Live Updates (Minimal)

Returns minimal information about the specified Bus Stop and a list of upcoming departures.

```jsx
GET /v1/bus/live-updates-minimal
```

### Query Parameters (optional)
| Name | Description | Type | Additional Information |
|------|-------------|------|-----------------------|
| `secondsAgo` | The number of seconds to look back for updates | Integer | Optional. Defaults to `5000`. |

#### Root
| Key | Type | Description |
|-----|------|-------------|
|`results`|Array|Array of `Update`s. See below.|

### Update
| Key | Type | Description |
|-----|------|-------------|
| `bus` | String | Unique identifier for the bus. |
| `line` | String |The bus service number, found on the LibertyBus timetable. |
| `category` | String | Type of vehicle (`Public Bus`, `School Bus`). |
| `latitude` | Decimal number | Current latitude of the bus. |
| `longitude` | Decimal number | Current longitude of the bus. |
| `bearing` | Integer | Compass bearing of the bus in degrees (0â€“359). |
| `direction` | String | Direction of the bus (`"inbound"` or `"outbound"`). |
| `time` | String (HH:MM:SS) | Local time when this position was recorded. |
| `age` | Integer | How many seconds ago this position was recorded. |

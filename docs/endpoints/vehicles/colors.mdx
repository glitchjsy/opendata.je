# Vehicle Colors

Returns a list of all distinct vehicle colours and the total amount of cars with that colour.

```jsx
GET /v1/vehicles/colors
```

### Query Parameters (optional)
| Param | Description |
|-------|-------------
| `firstRegisteredAfter` | Return vehicles first registered **after** the given date (supports `YYYY/MM/DD` or `YYYY-MM-DD`) |
| `firstRegisteredBefore` | Return vehicles first registered **before** the given date (supports `YYYY/MM/DD` or `YYYY-MM-DD`) |
| `firstRegisteredInJerseyAfter` | Return vehicles first registered **in Jersey after** the given date (supports `YYYY/MM/DD` or `YYYY-MM-DD`) |
| `firstRegisteredInJerseyBefore` | Return vehicles first registered **in Jersey before** the given date (supports `YYYY/MM/DD` or `YYYY-MM-DD`) |

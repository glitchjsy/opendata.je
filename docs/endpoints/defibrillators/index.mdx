# Defibrillators

Returns a list of all public defibrillators on the island.

```jsx
GET /v1/defibrillators
```

#### Root
| Key | Type | Description |
|-----|------|-------------|
|`results`|Array|Array of `Defibrillator`s. See example response.|

#### Defibrillator
| Key | Type | Description |
|-----|------|-------------|
|`id`|UUID|A unique ID to represent this defibrillator|
|`location`|String|A short description of the defibrillator location|
|`streetName`|String|The name of the street the defibrillator is on|
|`parish`|String|The parish the defibrillator is located in|
|`postCode`|String|The post code the defibrillator is located in|
|`padNumber`|Number|The defibrillator number|
|`latitude`|Number|The latitude of the defibrillator|
|`longitude`|Number|The longitude of the defibrillator|
|`notes`|String/null|Any notes|

# FOI Producers

Returns a list of all distinct *producers* of Freedom of Information requests.

```jsx
GET /v1/foi-requests/producers
```

### Response

This is a sample response.

```json title="Status: 200 (OK)"
{
  "results": [
    "the Freedom of Information office",
    "the Children, Young People, Education and Skills",
    "the Statistics Jersey",
    "the Climate Emergency",
    "the Bailiff's Chamber"
  ]
}
```

### Sources & Attribution
|Data|Source|
|----|------|
|All data|[FOI disclosure log on gov.je][foilog]|

*This information is obtained by scraping the FOI disclosure log on Gov.je every few hours in a way that is limited to the most recent entries only to reduce load on the site.*

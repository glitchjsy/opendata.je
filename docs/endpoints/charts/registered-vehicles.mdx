# Registered Vehicles

Returns total number of registered vehicles per year.

```
GET /v1/charts/registered-vehicles
```

:::info NOTE
This data is only used for [this chart](/charts/transport/registered-vehicles) and does not necessarily conform to the same design patterns as the rest of the API.
:::

Figures include all motor vehicles including commercial and private vehicles.

Figures up to 1993 include only taxed vehicles. Figures from 1994 onwards include all vehicles recorded on the DVS register at 31 December of each year.

# Monthly Rainfall

Returns information about monthly rainfall between 1894 and 2020.

```
GET /v1/charts/monthly-rainfall
```

:::info NOTE
This data is only used for [this chart](/charts/weather/monthly-rainfall) and does not necessarily conform to the same design patterns as the rest of the API.
:::

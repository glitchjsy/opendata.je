# Bus Passengers

Returns total number of bus passengers between 2019 and 2024.

```
GET /v1/charts/bus-passengers
```

:::info NOTE
This data is only used for [this chart](/charts/transport/bus-passengers) and does not necessarily conform to the same design patterns as the rest of the API.
:::
